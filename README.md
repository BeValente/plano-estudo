## Funcionalidades

| Secção | O que faz |
|---|---|
| **Dashboard** | Visão geral com progresso global, tarefas pendentes e conteúdos a estudar |
| **Matérias** | Adiciona disciplinas com cor, professor e slider de progresso manual |
| **Conteúdos** | Tópicos de estudo por semana, filtráveis por matéria e estado |
| **Horário** | Blocos semanais detalhados com hora, descrição e duração |
| **Calendário** | Vista mensal navegável com eventos, testes e entregas |
| **Tarefas** | Checklist com data de entrega e alerta de atraso |
| **Projetos** | Projetos práticos por matéria com barra de progresso |
| **Recursos** | Links, vídeos, artigos e ferramentas organizados por matéria |
| **Análise** | Estatísticas completas com círculos e barras de progresso |

## Tecnologias usadas

Este projeto foi construído com tecnologias web básicas — sem frameworks, sem bibliotecas externas, sem servidor:

- **HTML** — estrutura da página
- **CSS** — visual, animações e layout responsivo
- **JavaScript** — toda a lógica e interatividade
- **localStorage** — guarda os dados no browser do utilizador (sem base de dados)

## Estrutura do projeto

```
plano-estudo/
│
└── index.html      ← toda a aplicação num único ficheiro
```

O ficheiro `index.html` está dividido em 3 secções comentadas e detalhada:

```
1. CSS        → define o visual (cores, tamanhos, layout)
2. HTML       → define a estrutura (o que existe na página)
3. JavaScript → define a lógica (o que acontece ao clicar)
```

## Como usar

### Opção 1 — Ver online
Acede diretamente em: **[bevalente.github.io/plano-estudo](https://bevalente.github.io/plano-estudo/)**

### Opção 2 — Correr localmente
```bash

git clone https://github.com/BeValente/plano-estudo.git

# 2. Entra na pasta
cd plano-estudo

# 3. Abre o ficheiro no browser
# (duplo clique no index.html, ou arrasta para o browser)
```

Não precisas de instalar nada — é só abrir o ficheiro!

## Como os dados são guardados

Os dados ficam guardados no localStorage do teu browser — isto significa:

- Gratuito, sem servidor, sem conta
- Os dados persistem ao fechar e reabrir o browser
- Os dados ficam apenas no teu dispositivo
- Limpar os dados do browser apaga tudo
